Classe Color (Couleur)

Constructor

new Color(r, g, b, a)

Description:
  • Crée une nouvelle instance de couleur. La valeur alpha est optionnelle et par défaut à 1.
Properties:
Name Type Description
r number Valeur de Rouge
g number Valeur de Vert
b number Valeur de Bleu
a number Valeur d'Alpha
Example
// Pour un exemple complet d'utilisation de la classe Color,
// veuillez consulter le module footer.js.
const redColor = new Color(255, 0, 0);
const greenColor = Color.fromHex("#00FF00");
console.log(greenColor.toRGBA());  // "rgba(0,255,0,1)"
Parameters:
Name Type Default Description
r
g
b
a 1

Color

Methods

clamp() → {Color}

Description:
  • Limite les valeurs de couleur entre 0 et 255 pour RGB et entre 0 et 1 pour alpha.
Example
const color = new Color(300, -20, 100);
color.clamp();  // Résultat : r=255, g=0, b=100
Returns:
Type
Color

operate(operation, operand) → {Color}

Description:
  • Effectue une opération sur la couleur.
Example
const color = new Color(100, 100, 100);
color.operate('+', 50);  // Résultat : r=150, g=150, b=150

toRGBA() → {string}

Description:
  • Convertit la couleur en une chaîne RGBA.
Example
const color = new Color(255, 0, 0, 0.5);
console.log(color.toRGBA());  // "rgba(255,0,0,0.5)"

Type
