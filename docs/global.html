Methods

generate(stylesheet)

Description:
  • Génère des classes CSS dynamiquement en utilisant des mixins dynamiques.
Example
import { generate } from './generate.js';
import { StyleSheet } from '../Classes/StyleSheet.js';

const stylesheet = new StyleSheet();
generate(stylesheet);

// Dans ce cas, la fonction `generate` ajoutera les classes suivantes à l'instance `stylesheet` :
// .texte-1 { font-size: 1.2em; }
// .texte-2 { font-size: 1.2em; }
// ...
// .texte-6 { font-size: 1.2em; }
// .m-1 { margin: 10px; }
// .m-2 { margin: 20px; }
// ...
// .m-6 { margin: 60px; }
Parameters:
Name Type Description
stylesheet StyleSheet L'instance de la feuille de style à laquelle ajouter les classes.
